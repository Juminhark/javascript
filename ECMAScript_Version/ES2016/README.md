# new features in ECMAScript 2016(ES7)

- Exponentiation (\*\*)
- Exponentiation assignment (\*\*=)
- Array.includes()

## Exponentiation (\*\*)

```js
2 ** 10; // 1024

// x ** y produces the same result as Math.pow(x, y)
Math.pow(2, 10);
```

## Exponentiation assignment (\*\*=)

```js
5 **= 2; // 25
```
